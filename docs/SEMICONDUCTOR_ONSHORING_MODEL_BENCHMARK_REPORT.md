# Semiconductor Onshoring Model Benchmark

## Scope

Comparison of full DTE feedback routing, frozen-telemetry heterogeneous agents with identical hard constraints, and a static mean-telemetry expected-flow model without BOM or capacity dynamics.

- Policy cells: `60`
- DTE robust cells: `36`
- Frozen-agent robust cells: `16`
- Cells where feedback changes robust classification: `20`
- Static majority-onshore predictions: `60`
- Static majority predictions not robust under DTE: `24`

## Reading

The static model is a route-attractiveness benchmark, not a production model: it cannot represent BOM starvation, gate service limits, or adaptive telemetry. The frozen-agent model isolates the value of DTE feedback while preserving the same heterogeneous population and hard constraints.
