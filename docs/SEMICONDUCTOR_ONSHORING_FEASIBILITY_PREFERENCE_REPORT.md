If robust cells appear only after allocation pressure is introduced, the capacity-only result is confirmed: physical feasibility is necessary but does not select the domestic route. If no robust cells appear, the current topology requires structural change rather than stronger preference on existing edges. A flat tariff axis means the tariff is applied after route commitment on a serial corridor, where no competing outgoing edge exists.
